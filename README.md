# ST3S-Enhanced-GPT

## Features

- User can select the model's temperature
- Chat log is displayed in real-time
- User can clear the chat log
- User can report bugs

## Built With

- React
- Material UI
- OpenAI API
- Express
- react-icons
- body-parser
- cors
- morgan
